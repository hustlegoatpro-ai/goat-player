# Goat Player

A free, open-source web audio player for independent artists.

## Included sample tracks

- Changing
- Peace
- Rever
- Onward

## GitHub Pages

Upload every file in this package to the repository root.

Then open:

**Settings → Pages → Deploy from a branch → main → / (root)**
